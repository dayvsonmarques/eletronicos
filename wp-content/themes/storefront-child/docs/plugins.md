# Plugins obrigatórios

## WooCommerce
- Versão recomendada: mais recente estável
- https://br.wordpress.org/plugins/woocommerce/

## WooCommerce Stripe Gateway
- Versão recomendada: mais recente estável
- https://br.wordpress.org/plugins/woocommerce-gateway-stripe/

## WooCommerce Correios
- Versão recomendada: mais recente estável
- https://br.wordpress.org/plugins/woocommerce-correios/

## Como instalar

No painel WordPress, vá em **Plugins > Adicionar novo**, pesquise pelo nome do plugin, clique em **Instalar** e depois em **Ativar**.

Mantenha os plugins sempre atualizados para garantir segurança e compatibilidade.
